# Frontend

React + Vite SPA с локальным кэшем и визуализацией карт.

## Рекомендуемая структура

```text
frontend/
├── index.html
├── package.json
├── public/
├── src/
│   ├── app/
│   ├── pages/
│   ├── widgets/
│   ├── features/
│   ├── entities/
│   ├── shared/
│   ├── processes/
│   └── styles/
└── tests/
```

## Принцип

- UI и доменные сущности разделяются по слоям.
- Локальный кэш и оффлайн-режим живут отдельно от визуальных компонентов.
- D3/сплайны должны оставаться в слое визуализации, а не в бизнес-логике.
