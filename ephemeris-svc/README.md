# Ephemeris Service

Изолированный сервис для Swiss Ephemeris и всего, что связано с AGPL-слоем.

## Рекомендуемая структура

```text
ephemeris-svc/
├── Cargo.toml
├── src/
│   ├── lib.rs
│   ├── bin/
│   │   └── ephemeris-svc/main.rs
│   ├── ffi/
│   └── service/
└── tests/
```

## Принцип

- Сервис не должен тянуть бизнес-логику верхнего уровня.
- Вся публичная поверхность - через gRPC/FFI-границу.
- Бинарь можно деплоить отдельно и заменять независимо.
