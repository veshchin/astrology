# Proto

Версионированные protobuf-контракты для внутренних gRPC-вызовов.

## Рекомендуемая структура

```text
proto/
├── astro/
│   └── v1/
│       ├── api.proto
│       ├── core.proto
│       └── ephemeris.proto
├── buf.yaml
└── buf.gen.yaml
```

## Принцип

- Версия API фиксируется в пути пакета.
- Контракты не смешивают экспериментальные и стабильные сообщения.
- Генерация кода делается из одного источника правды.
